# Run DAT Kitchen

Kitchen inventory management, production tracking, and waste management system.

## Features

- 📦 **Inventory Count & Order Sheets** - Track stock levels and generate printable order sheets
- 🍳 **Production Entry** - Log daily production quantities
- 🗑️ **Waste Tracking** - Record and analyze waste
- 📊 **Manager Dashboard** - Performance analytics and insights
- 📈 **Executive View** - Charts, trends, and reports
- 📄 **Print Forms** - Print production and waste tracking sheets
- 💾 **LocalStorage** - No database required, all data stored in browser

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy to Production

### Vercel (Recommended)

```bash
vercel --prod
```

### PWA Installation

App can be installed on mobile devices:
1. Open in mobile browser
2. Tap "Add to Home Screen"
3. Launches like a native app

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- LocalStorage API
- Recharts for analytics
